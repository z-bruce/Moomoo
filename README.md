## Website uses Bootstrap 4

### User Data Structure

- User ID
- First Name
- Middle Name
- Last Name
- Email Address
- Phone Number
- Address - includes all the components that define an address
- Password
- Payment Method (Debit, Credit or Store Card)
- Card Number
- CVC Number

### Product Data Structure

- Item Name
- Serial Number
- Description
- Price
- Quantity
- Size/Weight
- Unit
- Type
- Aisle

### Order Data Structure

- Order ID
- Customer ID - associates an order with a User (as defined above)
- Total Price
- Order Status
- List of Products
